🚀 1. Importing Express
This imports the Express framework into your Node.js project. 
Express helps you create servers and APIs easily.

Example: 
const express = require("express");

🏗️ 2. Creating App (Server Instance)
"express()" creates a server instance
This "app" object is your main control center (routes, middleware, etc.)

Example: 
const app = express();

🌐 3. Creating a Route (/)
app.get()   → handles GET requests
"/"         → root URL (http://localhost:3000/)
(req, res)
req         → request from user
res         → response you send back

Example: 
app.get("/", (req, res) => {
    console.log("HelloWorld");
    res.send("HelloWorld");
});
